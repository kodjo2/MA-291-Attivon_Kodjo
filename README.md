# MA-291 - Initiation au JavaScript

Ce projet a été conçu pour vous préparer à l’évaluation pratique du module.

Les compétences travaillées sont les suivantes :

- Injecter (fetch) des données de type json dans une vue HTML.
- Réaliser des modifications de DOM grâce à de la programmation événementielle.
- Adapter un projet existant sur la base d’un cahier des charges.

# Modalités

| Critères            | Valeur     |
| :------------------ | :--------- |
| Temps à disposition | 5 périodes |

# Livraison

Créer une archive compressée répondant aux critères suivants :

- le répertoire `src`
- le répertoire `data`

Le projet doit se lancer via **Run** ou **Debug** sur le fichier `src/index.html`.
