# TASK 001 - Plus de snacks

## Objectif

Ajouter deux snacks à la liste existante dans `data/snacks.json`.

### Contraintes

- Chaque snack doit contenir `name`, `alt`, `description`, `price`, `imageUrl`.
- Les nouveaux éléments doivent s’afficher à la suite des snacks existants.
- Le chargement s’effectue via le bouton **Load snacks**.
