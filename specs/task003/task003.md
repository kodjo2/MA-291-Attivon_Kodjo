# TASK 003 - Les points de vente

## Objectif

Afficher, sous les snacks, une section listant les points de vente du campus.

## Contraintes techniques

- Les données proviennent du fichier `data/points-of-sale.json`
- La séquence d'appel doit être : `index.html -> main.js -> fetchSalesPoints.js`
- La structure HTML doit suivre le modèle fourni dans `specs/task003/index-sales-points.html`
- La feuille de style dédiée est `specs/task003/sales-points.css`
